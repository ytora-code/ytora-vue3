# 动态表格规则

基于 Naive UI `NDataTable` 的动态表格组件，通过列配置数组描述表头、单元格渲染和交互行为，实现“数据驱动页面”的表格渲染方式。

本文档定义该组件的字段结构、默认行为、插槽规则和推荐用法。

## 1 设计目标

该组件遵循以下目标：

1. 使用 JSON 数组描述列，而不是手写重复表格模板
2. 内置常见业务列类型，覆盖后台管理系统的大多数表格场景
3. 列级自定义优先通过 `slot` 或 `render` 扩展，而不是无限透传样式参数
4. 支持行数据类型约束，`schema` 与 `data` 可以在业务侧保持明确的 TS 语义
5. 对选择、排序、空态、尺寸、单元格更新等常用能力提供统一 API

---

## 2 表格列 JSON 结构

表格列由四部分属性组成：

1. 身份属性
2. 展示属性
3. 类型扩展属性
4. 渲染扩展属性

### 2.1 身份属性

1. `key`：列唯一标识，未传时优先取 `dataKey`
2. `dataKey`：当前列绑定的字段名，`selection` 和 `index` 列可省略
3. `title`：表头标题
4. `type`：列类型，未传时默认为 `text`

**列类型**：支持以下内置类型：

1. `selection`：多选列，通常放在最左侧
2. `index`：序号列，默认从 `1` 开始
3. `text`：普通文本列，默认单行省略
4. `number`：数字列，默认右对齐
5. `date`：日期列，通常配合 `formatter` 使用
6. `image`：图片列，渲染缩略图
7. `link`：链接列，点击跳转 URL
8. `tag`：标签列，字符串按中英文逗号切分后展示多个标签
9. `switch`：开关列，适合行内启停状态
10. `slot`：纯插槽列，插槽名默认为列 `key`

### 2.2 展示属性

1. `width`：列宽，适合固定宽度列
2. `minWidth`：最小列宽，适合自适应列
3. `align`：对齐方式，支持 `left`、`center`、`right`
4. `fixed`：固定列，支持 `left`、`right`
5. `visible`：是否可见，默认 `true`
6. `ellipsis`：是否省略并显示 tooltip
7. `className`：列 class 名
8. `sorter`：是否启用排序
9. `emptyText`：空值时的占位文案，默认 `-`

### 2.3 类型扩展属性

1. `formatter`：格式化输出，适合 `text`、`number`、`date`
2. `indexStart`：序号列起始值
3. `imageWidth` / `imageHeight`：图片列尺寸
4. `imageAlt` / `imageFallbackSrc` / `imagePreviewDisabled`：图片列补充配置
5. `linkText` / `linkTarget` / `linkRel`：链接列补充配置
6. `tagMap`：标签映射，如状态值转标签文案与颜色
7. `tagSplitPattern`：标签拆分规则，默认支持 `,` 和 `，`
8. `switchCheckedValue` / `switchUncheckedValue`：开关列映射值
9. `switchCheckedText` / `switchUncheckedText`：开关文案
10. `switchDisabled`：开关列是否禁用

### 2.4 渲染扩展属性

1. `render`：完全自定义渲染函数

渲染优先级如下：

1. `render`
2. 具名插槽
3. 内置列渲染

## 3 默认行为

为了降低使用成本，组件采用以下默认行为：

1. 未传 `type` 时按 `text` 处理
2. 未传 `key` 时优先使用 `dataKey`
3. `selection`、`index`、`image`、`tag`、`switch` 默认居中
4. `number` 默认右对齐
5. `text`、`number`、`date`、`link`、`tag` 默认开启省略提示
6. `link` 默认新窗口打开，并附带安全 `rel`
7. `switch` 默认值映射为 `true / false`
8. 组件空态默认显示“暂无数据”

## 4 表格组件 API

### 4.1 Props

1. `data`：表格数据数组
2. `schemas`：列配置数组
3. `rowKey`：行唯一键，支持字段名或函数
4. `checkedRowKeys`：选中行 key 数组
5. `loading`：加载状态
6. `size`：表格尺寸
7. `striped`：是否斑马纹
8. `bordered`：是否显示边框
9. `singleLine`：是否使用单行分隔样式
10. `singleColumn`：是否显示纵向分隔线
11. `remote`：是否远程排序/分页
12. `pagination`：透传 Naive UI 分页配置，传 `false` 表示不启用
13. `scrollX`：横向滚动宽度
14. `minHeight` / `maxHeight`：表格垂直滚动高度
15. `minWidth` / `maxWidth`：表格容器宽度边界
16. `emptyDescription`：空态文案

### 4.2 Events

1. `update:data`：当 `switch` 等可交互列修改行值时触发
2. `update:checkedRowKeys`：勾选项变化时触发
3. `selection-change`：返回选中 key 和选中行
4. `cell-change`：单元格值变更事件
5. `row-click`：行点击事件
6. `sort-change`：排序变化事件

### 4.3 Expose

1. `scrollTo`：滚动到指定位置
2. `getCheckedRows`：获取当前选中行数据

---

## 5 插槽规则

动态表格支持以下插槽：

1. `empty`：自定义空态
2. `#${key}`：列插槽，名称与列 `key` 一致

列插槽参数如下：

```ts
{
  ;(row, rowIndex, schema, value, setValue)
}
```

其中 `setValue` 可用于更新当前行该列的值，适合在 `switch`、操作列、组合展示列中复用。

## 6 推荐用法

### 6.1 基础示例

```vue
<script setup lang="ts">
import { ref } from 'vue'
import type DynamicTableSchema from '@/components/table/type/DynamicTableSchema'

interface UserRow {
  id: number
  userName: string
  status: number
  avatar: string
}

const tableData = ref<UserRow[]>([{ id: 1, userName: 'admin', status: 1, avatar: '/avatar.png' }])

const schemas: DynamicTableSchema<UserRow>[] = [
  { type: 'selection', key: 'selection', width: 48 },
  { type: 'index', key: 'index', title: '序号', width: 64 },
  { title: '用户名', dataKey: 'userName', minWidth: 180 },
  {
    title: '状态',
    dataKey: 'status',
    type: 'tag',
    width: 120,
    tagMap: {
      '1': { label: '正常', type: 'success' },
      '0': { label: '禁用', type: 'error' },
    },
  },
  {
    title: '头像',
    dataKey: 'avatar',
    type: 'image',
    width: 96,
    imageWidth: 36,
    imageHeight: 36,
  },
]
</script>

<template>
  <Table v-model:data="tableData" :schemas="schemas" row-key="id" />
</template>
```

### 6.2 操作列插槽

```vue
<Table :data="tableData" :schemas="schemas" row-key="id">
  <template #actions="{ row }">
    <n-button text type="primary">编辑 {{ row.userName }}</n-button>
  </template>
</Table>
```

对应列配置：

```ts
{
  key: 'actions',
  title: '操作',
  type: 'slot',
  width: 180,
  fixed: 'right'
}
```
